# Snyk 시작하기

## Snyk 계정 생성&#x20;

1. [https://snyk.io/](https://snyk.io)에서 **SIGN UP** 클
2. 로그인하거나 가입방법을 선택
3. 메시지가 표시되면 로그인 진행 또는 계정 생성 진행
4. Snyk 계정 생성 시 다음 내용을 수행할 수 잇습니다.
   1. 간단한 정보 입력 후 프로젝트 통합 진행
   2. [https://app.snyk.io](https://app.snyk.io)에서 Snyk 대시보드 액세스
