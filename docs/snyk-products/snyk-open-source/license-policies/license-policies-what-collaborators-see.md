# 라이선스 정책: collaborator가 보는 내용

예약된 다음 테스트가 실행되면 새로 할당된 정책 또는 정책 수정 사항이 적용됩니다. 조직의 모든 라이선스에 대해 다음과 같은 일이 발생합니다.
