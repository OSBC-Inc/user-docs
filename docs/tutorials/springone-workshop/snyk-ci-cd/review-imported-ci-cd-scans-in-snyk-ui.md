## Snyk CI 세부 결과

상세 보기에서는 소프트웨어 아티팩트에 대해 동일한 세부 정보를 제공하지만 SDLC에서는 시간이 다릅니다. 이 예에서는 소스가 표시한 대로 CI 파이프라인 중에 스캔 결과를 캡쳐했습니다. CI 결과를 모니터링하기로 선택했기 때문에 수정 사항이 제공되면 Snyk에서 알림을 제공합니다.
