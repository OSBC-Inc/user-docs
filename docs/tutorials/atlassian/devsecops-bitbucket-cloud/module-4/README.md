## 배경

1. 애플리케이션 종속성의 취약성에 대한 **수정 풀 요청** 제출
2. 컨테이너 이미지에 **기본 이미지 업그레이드** 권장 사항 적용
3. Kubernetes 매니페스트에서 **보안 컨텍스트** 정의
