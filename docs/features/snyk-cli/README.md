# Snyk CLI

Snyk CLI는 개발 워크플로우에 [Snyk의 기능](https://docs.snyk.io/introducing-snyk#snyk-products)을 제공합니다. CLI를 로컬로 실행하거나 CI/CD 파이프라인에서 실행하여 보안 취약성 및 라이센스 문제를 비롯한 보안 문제를 검색할 수 있습니다. 다음은 Snyk CLI 테스트 명령 출력의 예를 보여 줍니다.

![Snyk CLI 테스트 명령 출력](<../../.gitbook/assets/snyk cli 1.png>)

오픈 소스용 Snyk CLI 검색은 Java, JavaScript, Python, PHP, Ruby, Scala, .NET, Golang 및 C/C++를 비롯한 많은 언어와 도구를 지원합니다. 자세한 내용은 [오픈 소스 언어 및 패키지 관리자 지원](../../snyk-products/snyk-open-source/language-and-package-manager-support/)을 참조하십시오.

또한 Snyk CLI는 다음을 지원합니다:

* [Snyk 코드 검색](../../snyk-products/snyk-code/cli-for-snyk-code/)
* Docker 스캔을 포함한 [Snyk 컨테이너 스캔](../../snyk-products/snyk-container/snyk-cli-for-container-security/)
* Terraform and Kubernetes(K8s) 프로젝트를 포함한 [코드 스캔으로서의 Snyk Infrastructure](../../snyk-products/snyk-infrastructure-as-code/snyk-cli-for-infrastructure-as-code/)

자세한 내용은 [지원되는 언어](../../introducing-snyk/)를 참조하십시오.
